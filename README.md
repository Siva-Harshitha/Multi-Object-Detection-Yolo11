YOLO11 Multi-Object Tracking:

Upload a video → Get real-time object detection with tracking boxes and labels.

Watch cars, trucks, and other objects get detected and followed across your video with persistent ID numbers. Perfect for traffic analysis, surveillance, or any video you want to analyze.

Features:

📹 Upload any video (MP4, MOV, AVI, etc.)
🎯 Detect and track multiple objects automatically
🏷️ Each object gets a unique tracking ID
💾 View detections frame-by-frame
🎨 Beautiful, dark theme interface
⚡ Powered by YOLO11 (fast & accurate)

How it works:

Upload a video through the web interface
It analyzes the video and detects objects
Watch each frame with bounding boxes showing what was detected
Click "Play" to see smooth video playback with all detections overlaid

Tech Stack:

Python + Flask (backend)
YOLO11 (object detection & tracking)
HTML/CSS/JavaScript (frontend)
OpenCV (video processing)
